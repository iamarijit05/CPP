# Object-Oriented Programming in C++

This repository contains C++ programs demonstrating the core concepts of Object-Oriented Programming (OOP). It serves as a learning resource and reference for understanding how OOP principles are implemented in C++.

## 📚 Topics Covered

* Classes and Objects
* Constructors and Destructors
* Copy Constructors
* Dynamic Initialization of Objects
* Static Data Members and Static Functions
* Friend Functions and Friend Classes
* Operator Overloading
* Inheritance

  * Single Inheritance
  * Multiple Inheritance
  * Multilevel Inheritance
  * Hierarchical Inheritance
  * Hybrid Inheritance
* Polymorphism

  * Function Overloading
  * Function Overriding
  * Virtual Functions
* Abstract Classes
* Pointers to Objects
* Object Arrays
* Dynamic Memory Allocation
* Templates
* Exception Handling
* File Handling using Classes

## 🛠️ Requirements

* C++17 or later
* GCC/G++ Compiler
* VS Code, CodeBlocks, CLion, or any C++ IDE

## 🚀 Compile and Run

Compile:

```bash
g++ filename.cpp -o program
```

Run:

```bash
./program
```

For Windows:

```bash
g++ filename.cpp -o program.exe
program.exe
```

## 🎯 Purpose

The goal of this repository is to strengthen OOP concepts in C++ through practical examples and implementations.
